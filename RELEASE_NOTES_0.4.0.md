# Karl Jr 0.4.0 Release Notes

We're excited to announce Karl Jr 0.4.0, featuring a powerful new accessibility checker to help you create more accessible and readable content for SF.gov!

## What's New

### Accessibility Checker

Run comprehensive accessibility tests on any SF.gov page with a single click. The new accessibility checker helps you identify and fix common issues before publishing.

**What it checks:**

- **Heading structure** - Ensures headings follow proper hierarchy (H1 → H2 → H3) so screen readers can navigate your page correctly
- **Image alt text** - Identifies images missing alternative text descriptions
- **Link accessibility** - Flags vague link text like "click here" and raw URLs pasted into content
- **Table accessibility** - Verifies tables have captions and proper header rows/columns
- **Video accessibility** - Checks that videos include both captions and transcripts
- **Readability score** - Calculates grade level using the Automated Readability Index to help you write for an 8th grade reading level

**Visual feedback:** Issues are highlighted directly on the page so you can quickly find and fix them.

### Hemingway App Integration

Get additional writing suggestions with seamless Hemingway App integration.

**How it works:**
1. Click "Get help in Hemingway App" after running accessibility tests
2. Page text is automatically copied to your clipboard
3. Follow the step-by-step instructions to paste into Hemingway
4. Click "Open Hemingway App" when you're ready
5. Compare scores and get suggestions for simpler sentences and word choices

The instructions stay visible even when you switch tabs, so you can work at your own pace.

### Readability Improvements

- Grade levels now display as whole numbers (no decimals)
- Reading levels above college are labeled "Post-graduate"
- Clearer recommendations for improving readability
- Structure analysis identifies long paragraphs and sections that need headings
